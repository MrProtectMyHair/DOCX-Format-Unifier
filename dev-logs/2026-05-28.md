## 补充 — 段落表单式填充（保留下划线格式）

### 问题
模板段落使用下划线/空白标记填入位置（如 `本人______，身份证号 ________`），当前 `_set_paragraph_text` 将所有文本合并到第一个 run，导致模板的下划线格式和 run 结构全部丢失。

### 实现
- 新增 `_is_blank_run()`：判断 run 是否为空白占位（纯下划线 `_` 或带下划线格式的纯空白）
- 新增 `_fill_form_paragraph()`：表单式段落智能填充
  - 分类 run 为 fixed（固定文字）和 slot（空白槽），合并连续空白 run
  - 用 fixed text 作为锚点在输入文本中定位，提取填充值
  - 填充值替换空白 run 文本，保留原 run 的所有格式属性（underline、font name/size、bold）
  - 若段落不含空白 run 或全为空白，降级为 `_set_paragraph_text` 行为
- 修改 `_replace_paragraphs()`：将所有 `_set_paragraph_text` 调用替换为 `_fill_form_paragraph`

### 验证
- Reference_04 工作说明：姓名、性别、身份证号正确填入，身份证号保留下划线格式
- Reference_01 审批表 + Reference_05 分配表回归：段落+表格匹配无退化
