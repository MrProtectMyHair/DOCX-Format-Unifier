## 补充 — 段落表单式填充（保留下划线格式）

### 问题
模板段落使用下划线/空白标记填入位置（如 `本人______，身份证号 ________`），当前 `_set_paragraph_text` 将所有文本合并到第一个 run，导致模板的下划线格式和 run 结构全部丢失。

### 实现
- 新增 `_is_blank_run()`：判断 run 是否为空白占位（纯下划线 `_` 或带下划线格式的纯空白）
- 新增 `_fill_form_paragraph()`：表单式段落智能填充
  - 分类 run 为 fixed（固定文字）和 slot（空白槽），合并连续空白 run
  - 用 fixed text 作为锚点在输入文本中定位，提取填充值
  - 填充值替换空白 run 文本，保留原 run 的所有格式属性（underline、font name/size、bold）
  - 若段落不含空白 run 或全为空白，降级为 `_set_paragraph_text` 行为
- 修改 `_replace_paragraphs()`：将所有 `_set_paragraph_text` 调用替换为 `_fill_form_paragraph`

### 验证
- Reference_04 工作说明：姓名、性别、身份证号正确填入，身份证号保留下划线格式
- Reference_01 审批表 + Reference_05 分配表回归：段落+表格匹配无退化

## 补充 — 段落匹配修复（标签剥离 + 锚点匹配 + 日期行间距）

### 问题
Input_03 劳务合同书 + Template_03 劳务合同书：
1. "第二条""第五条"标签前缀被错误剥离
2. "课酬"中 3000 下划线丢失，"税后 大写"重复出现
3. 日期行间距与模板不一致

### 根因和修复
1. `_strip_label_if_template_has_none`：`_extract_label` 去掉了空格，但比较时用原始模板文本（有空格）→ 误判 → 增加去空格归一化比较
2. `_fill_form_paragraph` 锚点搜索用精确 `str.find`，模板 `"  税后  大写："` 与输入 `" 税后 大写："` 空格数不同 → 匹配失败 → 增加空格模糊回退（正则 `\s*`）
3. 无 blank run 段落降级到 `_set_paragraph_text`，破坏模板日期段的精细 run 结构 → 去空格文本一致时保留模板 run 结构
4. `str.find` 回退时 `match_end` 使用了模板文本长度而非实际匹配长度 → 修正为 `m.end()`

### 关于 3000 下划线
模板中 `"_______"` 占位符的 underline=None。表单填充保留模板格式，若需要填入值带下划线，需在模板中给占位符加下划线格式。

## 补充 — 电话下划线数字消失修复

### 问题
Input_03 + Template_03，乙方 form 段落中 `"电话：___________"` 填入值为空，电话号被追加到文档末尾。

### 根因
1. **尾部截断空格误判**：最后一个 slot 填入值后，尾部截断逻辑收集后续空格 run，在 fill value 中找到空格在位置 0 → 截断为空。修复：`trailing.lstrip()` 后再搜索。
2. **电话在独立输入段落**：输入 P[3] "电话： 13777557011" 与 P[2] form 正文分离，"电话：" 锚点未在 P[2] 中找到。修复：`_replace_paragraphs` 中，若未匹配段落的 label 在模板中存在但不在当前 input 中，合并未匹配段落到 input text 后再调用 `_fill_form_paragraph`。

## 补充 — 表格文档保存 PermissionError 修复

### 问题
GUI 中用 Input_01_审批表 / Input_05_分配表（含表格）转换时 `doc.save()` 报 PermissionError。纯段落文档（Input_03/04）正常。

### 根因
`shutil.copy2` 在目标路径创建 .docx 后，Windows Explorer 检测到新文件并打开生成缩略图。表格文档处理时间更长（替换 27 个单元格），Explorer 有足够时间锁住文件 → `doc.save()` 失败。

### 修复
`generate_output` 改为：复制模板 → 系统临时目录（`tempfile.mkstemp`）→ 修改 → 保存 → `shutil.move` 到目标路径。临时目录不被 Explorer 监控，避免了文件锁。
