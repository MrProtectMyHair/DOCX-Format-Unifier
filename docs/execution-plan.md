## 已实现的核心特性

### 段落匹配
- 标签匹配（Phase 1）→ 复合段落检测（Phase 1.5）→ 位置回退（Phase 2）
- 47 个常见中文字段标签 + 下划线核心标签归一化比对
- 未匹配段落的 label 智能合并

### 表格匹配
- 列头引导匹配 + 精确唯一文本匹配 + 标签全局匹配
- **多表格支持**（遍历所有 table 对）

### 空值填充（writer.py）
- 标签领地填充：每个标签管辖其到下一个标签之间的空单元格
- 列头引导填充：纯数据行向上找表头，按列映射+行偏移填入
- 汇总标签（合计/总计/小计）仅填紧邻一格
- 签章/行政标签（8个）跳过保护

### 表单式段落填充
- 自动识别模板中的下划线/空白占位（`_is_blank_run`）
- 固定文本锚点定位 → 提取输入值 → 保留原 run 格式（underline, font, bold）
- 空格模糊回退（正则 `\s*`）

### 文件生成
- 临时文件放在输出目录（避免 Windows Explorer / 杀软锁文件）
- `copy2` 后 `os.chmod` 移除只读
- uuid 文件名，避免预创建

### GUI
- customtkinter Professional Blue 主题
- 全中文 UI（标签、按钮、日志、对话框）
- 手动映射弹窗（可折叠行组、下拉选择）
